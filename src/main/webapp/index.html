1. På en ny side får man status kode 200. Men hvis du reloader får man 304, fordi den ligger i din cache, så bliver man bare redirected.

2. Både billedet og style sheetet bliver hentet ved reload. Hvor de virker på samme måde, ved at blive serveret ved memory cache efter et reload. I både response of request headeren kan der ses en connection header. Denne header siger “keep-alive” hvilket nok er grunden til at vi kan bruge memory cachen til at hente noget vi allerede har downloadet før.
